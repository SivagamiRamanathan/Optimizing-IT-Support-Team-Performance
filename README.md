**Section 1 – Title**
**Optimising IT Support Team Performance Using Analytics**
This project focuses on analyzing IT support team data to evaluate performance, identify inefficiencies, and improve overall service quality.

**Section 2 – Problem Statement**
IT support teams handle multiple tickets daily, including technical issues, system errors, and user complaints.
However, organizations often face challenges such as:
Delayed ticket resolution
Uneven workload distribution
Low customer satisfaction
Lack of performance tracking
There is a need for a data-driven solution to monitor Key Performance Indicators (KPIs) and improve IT support team efficiency.

**Section 3 – Technical Support Dataset Description**
The Technical Support Dataset contains service ticket records generated by an IT support team. The data represents real-time operational activities including ticket creation, assignment, resolution, and customer feedback.
This dataset is used to analyze team efficiency, service quality, and performance trends.

 Column Name                  Description                                              
 Ticket_ID                    Unique identifier for each support ticket                
 Issue_Category               Type of issue (Hardware, Software, Network, Login, etc.) 
 Priority_Level               Priority of the ticket (Low, Medium, High, Critical)     
 Ticket_Status                Current status (Open, In Progress, Resolved, Closed)     
 Assigned_Engineer            Name/ID of the support team member handling the ticket   
 Department                   Department that raised the issue                         
 Created_Date                 Date when ticket was raised                              
 First_Response_Time          Time taken for initial response                          
 Resolution_Date              Date when ticket was resolved                           
 Resolution_Time (Hours)      Total time taken to resolve the issue                    
 SLA_Status                   Whether SLA was met or breached                          
 Customer_Satisfaction_Score  Feedback rating given by user (1–5)        

 **Data Processing Steps:**

Removed missing or null values
Standardized date formats
Cleaned inconsistent category names
Created calculated columns for Resolution Time
Verified data accuracy

**Purpose of Dataset**

The dataset is used to:
Measure IT support performance
Track ticket resolution efficiency
Monitor SLA compliance
Analyze workload distribution
Evaluate customer satisfaction trends
This dataset forms the foundation for building KPIs and dashboards in Power BI.
